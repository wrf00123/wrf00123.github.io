使用帮助

快速开始

1. 从左侧面板选择图标来源：上传图片、选择预设图标或输入文字

2. 在中央预览区查看效果，拖拽可移动图标

3. 在右侧面板调整尺寸、形状、背景和滤镜

4. 选择导出尺寸和格式，点击下载

设置文件夹图标（Windows）

1. 下载 ICO 格式的图标文件

2. 右键文件夹 → 属性 → 自定义 → 更改图标

3. 选择下载的 .ico 文件，确定即可

或者：下载 desktop.ini，将其与图标文件放入目标文件夹，重启资源管理器

设置文件夹图标（macOS）

1. 下载 PNG 格式图标（512px 最佳）

2. 双击打开图片，按 CmdA 全选，按 CmdC 复制

3. 右键文件夹 → 显示简介 → 点击左上角图标 → 按 CmdV 粘贴

快捷键

CtrlZ 撤销操作

CtrlY 重做操作

+ / - 放大/缩小预览

0 适应画布

CtrlS 导出图标

关于 ICO 格式

ICO 是 Windows 图标格式，一个 ICO 文件可包含多个尺寸的图标，系统会根据显示场景自动选择最合适的尺寸。

导出的 ICO 文件包含：16×16、32×32、48×48、64×64、128×128、256×256 六种尺寸。
